## Management

Add sub apps,
```
git submodule add <remote_url> <destination_folder>
```

Get sub apps,
```
git submodule update --init --recursive
```

Fist, fetch submodule commits individually then,

Update sub apps,
```
git submodule update --remote --merge
```
